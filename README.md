## Kurulum (Termux)
Termux terminalini açın ve şu komutları sırayla kopyalayıp yapıştırın: git clone https://github.com/hunterofevil6504/HunterTool-v2.0.git

```bash
pkg update && pkg upgrade -y
pkg install git -y
git clone [https://github.com/hunterofevil6504/HunterTool-v2.0.git](https://github.com/hunterofevil6504/HunterTool-v2.0.git)
cd HunterTool-v2.0
chmod +x setup.sh
bash setup.sh


çalıştırma kodu:
cd ~/hunter-tool
bash hunter.sh
